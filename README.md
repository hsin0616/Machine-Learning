Course Information: https://course.ntu.edu.tw/courses/113-1/40152

This repository contains my coursework for Machine Learning (113-1: EE5184 / 921 U2620), a 4-credit course held every Tuesday from 9:10 AM to 1:10 PM.
- The first three hours are dedicated to lectures (9:10–12:10), followed by a TA session in the final hour (12:20–1:10).
- The course is taught in English, and all homework and exams are also conducted in English.
- Grading is based on a balanced structure: 30% midterm, 30% final exam, 30% homework, and 10% classroom participation.
- Prerequisites include Calculus, Linear Algebra, and Probability.
